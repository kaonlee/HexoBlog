Layout設計資源(1)

Created2023-06-24|Updated2023-06-24
|Post View:

近期完成了 ASP.NET MVC5 的課程，正在進行期中專題的準備。
期中專題我們是需要為我們小組設計的主題進行後台功能的開發，UI並不是本次專題的重點（助教建議使用網路上現有的模板）。
儘管最後我決定使用自己設計成想要的樣式，但還是藉此來記錄一下好用的網路資源。

Start Bootstrap

  • Start Bootstrap
  • https://startbootstrap.com/

ASP.NET MVC 預設就有掛上 Bootstrap 套件，所以只要上網搜尋 Bootstrap template 就可以看到一些不錯的資源！
要注意的是 Bootstrap 每個版本間的使用方式不大一樣。如我們現在使用的是 Bootstrap 5.3.0 版本，如果模組使用其他版本的話，就要記得進行微調。

Start Bootstarp 是助教推薦的網站，上面有 後台 跟 前台 的範本。就算是沒有要直接使用，也是有其參考價值。

W3 Layout

  • W3 Layout
  • https://w3layouts.com/templates/

W3 Layout 上面不全是 Bootstrap 的範本，且免費的資源也較少。但當作作品看看也是蠻有趣的！

CodePen

  • CodePen
  • https://codepen.io/your-work

CodePen 上可以查到許多靈感，也可以看到很多跟自己一樣是初學者程度的作品，可以有效的增加信心 XD。
這次的Layout我便參考 CodePen 的元件，了解作者是怎麼做出一些效果並且加強成自己想要的樣式。覺得蠻讚的。有事沒事上去逛也是非常的有幫助！
