Created2023-05-31
正在測試 Hexo, 原本我的 css 都跑不出來。後來發現抓不到的原因是因為在部屬檔案 _config.yml 中的 #URL > url 中所填入的頁面錯誤，例如我的網址是 https://kaonlee.github.io/HexoBlog/ 而非 https://kaonlee.github.io/ ，這就要注意不要寫錯了！ 部署參考文件 Hexo Official Document 中文教程 Bucky’s
